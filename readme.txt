RB Theme Helper is used for common theme options.

== Description ==

This plugin add generate 3 common options color, typography & switch. It depends on the plugin Kirki Customizer Framework.

== Installation ==

### From your WordPress dashboard

= Visit =

Plugins -> Add New

= Search =

for "RB Theme Helper"

= Install and Activate =

"RB Theme Helper" from your Plugins page
